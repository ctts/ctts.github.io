    # 为什么使用 jsx，它有什么好处

    1. 让 js 中的 html 代码更加直观
    2. 在使用 jquery 的时代，html 和 js 的代码是分离开放在不同页面的，这样我们在构建页面时，需要把 UI 逻辑和 js 逻辑分割开，但它们本质是其实是有强耦合性的，分离开思考可能会影响我们思考的逻辑。jsx 让 ui 逻辑和 js 逻辑合为一体，极大地优化了代码逻辑和清晰度。
